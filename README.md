# resolve-swagger-refs - Resolves Swagger 2.0 definition references

Usage:

```javascript
var resolveRefs = require('resolve-swagger-refs');
var definition = ... // Swagger 2.0 definition

var resolvedDefinition = resolveRefs(definition);
```

See also `/demo`.
